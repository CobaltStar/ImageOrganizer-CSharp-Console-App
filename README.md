# ImageOrganizer-CSharp-Console-App
Windows Application that organizes your images in a directory

## Installation:
1. Run the installer.
2. Specify any settings you would like.
3. All set!

## To run:
1. Right a directory to open the Windows Context Menu
2. Click "Organize Images"
3. Confirm that everything is being converted properly
4. All Set!

Features:
- Dialog box which tell you what your changes will be
- Update logs to track what files have been modified and when
- Program currently renames every file to the directory name and generates an ID based on time the file was created
